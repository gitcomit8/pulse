## Pulse - A social media app 

**Features:**

* User Accounts
* Post Feed
* Likes and Comments
* Friend System
* Notification
* RT chat

**Platforms:**

* Android
* Web

**Technology Stack:**

* **Backend:** NodeJS
* **Frontend:** ReactJS
* **Database:** PostgreSQL

**Priority:**

1. Database
2. Backend
3. Frontend
